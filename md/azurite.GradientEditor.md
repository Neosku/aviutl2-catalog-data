# Gradient Editor

[AviUtl2](https://spring-fragrance.mints.ne.jp/aviutl/) 用のグラデーションエディタプラグイン

## 動作環境

`beta36` 以降必須（`beta39` で動作確認済み）。

## 付属するスクリプト

本プラグインには以下のスクリプトが付属しています。

- 多色グラデーション（`MutliGradient@GradientEditor`）
- グラデーションマップ（`GradientMap@GradientEditor`）

グラデーションエディタ（`GradientEditor.aux2`）自体はこれらを直感的に操作するためのエディタに過ぎず、単体でグラデーション加工を施すことはできません。基本的には上記のスクリプトをオブジェクトに適用し、そのスクリプトのパラメータをグラデーションエディタ上で編集することになります。

>[!NOTE]
スクリプト単体でも使うことはできますが、グラデーションエディタ上で編集することを前提とした作りになっているため推奨しません。
