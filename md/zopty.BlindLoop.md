# 導入手順

1. `BlindLoop.anm2` をダウンロード
2. アプリケーションデータフォルダ内の `Script` フォルダに移動
3. 起動時に `このプラグイン・スクリプトを信頼して使用する` をクリック
4. `クリッピング` にエフェクトとして追加されます

# 使用方法

| パラメータ | 説明 | 既定値 | 範囲 |
| :---: | :---: | :---: | :---: |
| 割合 | ブラインドの開閉度<br>100で全開、0で全閉 | 50 | 0 ~ 100 |
| 幅 | ブラインドの間隔 | 30 | 1 ~ 1000 |
| 角度 | ブラインドの角度 | 60 | -1800 ~ 1800 |
| 位相 | ブラインドのズレ | 0 | -360 ~ 360 |
