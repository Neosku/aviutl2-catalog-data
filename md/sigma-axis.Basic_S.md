# Basic\_S パッケージ

## 概要

多機能なスクリプトセットです。角丸矩形やスーパー楕円といったカスタムオブジェクトの生成から、変形・配置・色調補正・クリッピングなどの高度なフィルタ効果まで、動画編集に役立つ汎用的な機能が多数収録されています。

## 使用方法

  * **カスタムオブジェクト**: タイムライン上で右クリック → メディアオブジェクトの追加 → **Basic\_S\\図形** → **{オブジェクト名}**
  * **フィルタ効果**: タイムライン上のオブジェクトを選択 → フィルタ効果の追加 → **Basic\_S{カテゴリ}** → **{効果名}**
