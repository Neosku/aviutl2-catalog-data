# ColoredCollar\_S

## 概要

オブジェクトの境界部分の色を引き延ばして、縁取りを作成するAviUtl ExEdit 2用のスクリプトです。

通常の「縁取り」とは異なり、縁部分を半透明にしたり、好きな色を薄く乗せてなじませたりすることも可能です。

## 使用方法

オブジェクトを選択し、エフェクト追加メニューから以下のように選択してください。

フィルタ効果の追加ボタン → **装飾** → **ColoredCollar\_S**

## 項目一覧

| 項目名 | 説明 |
| :--- | :--- |
| **サイズ** | 縁取りの太さ（幅）を指定します。 |
| **ぼかし** | 縁取り部分のぼかし具合を調整します。 |
| **透明度** | 縁取り部分（引き延ばした部分）の透明度を指定します。値を上げると縁が透明になります。 |
| **縁色** | 縁部分に乗せる色を指定します。「色の濃さ」と組み合わせて使用します。 |
| **色の濃さ** | 設定した「縁色」をどれくらい強く反映するかを指定します。<br>値を0にすると元の画像の色がそのまま引き延ばされ、値を上げると指定した「縁色」に近づきます。 |
| **前景透明度** | 縁取りの中にある「元のオブジェクト」自体の透明度を指定します。<br>縁だけを残して中身を消したい場合などに使用します。 |
| **PI** | パラメータインジェクション（上級者向け）。<br>スクリプト記述によりパラメータを一括制御するための項目です。通常は空欄で使用します。 |
