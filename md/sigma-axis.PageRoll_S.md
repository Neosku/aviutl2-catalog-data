# PageRoll_S

## 概要

画像を紙のように丸めて転がすアニメーション効果と、その動きを利用したシーンチェンジ効果を追加するスクリプトです。めくる方向・太さ・裏面画像などの調整が可能です。

## 使用方法

* フィルタ効果
  オブジェクト追加ボタン → **変形** → **PageRoll_S**
* シーンチェンジ
  オブジェクト追加ボタン → **シーンチェンジ** → **PageRoll_S(シーンチェンジ)**
