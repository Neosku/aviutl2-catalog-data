# ObjectMotionBlur_LK

## 概要

オブジェクトの動きに合わせて自然な**モーションブラー（動きぼかし）**をかけるスクリプトです。
位置や回転、拡大縮小などの動きを解析して、滑らかで映画的な残像表現を再現します。
AviUtl標準のモーションブラーよりも高精度で、動きの速い映像にも対応します。

---

## 使用法

オブジェクト追加ボタン → **ブラー** → **ObjectMotionBlur_LK**

---

## 使い方

このスクリプトは、オブジェクトの移動・回転・拡大縮小の変化量をもとにして**フレーム間の動きを補間し、ブラーをかける**仕組みです。
たとえば、オブジェクトを速く動かすと、その動きに沿った残像が表示されます。

---

## パラメーター一覧


| 項目名                   | 説明                                                         |
| :-------------------- | :--------------------------------------------------------- |
| **Shutter Angle**     | ブラーの強さ（幅）を設定します。360°で1フレーム分の動きと等しく、数値が大きいほどブラーが強くなります。     |
| **Sample Limit**      | ブラーの描画精度を設定します。値を上げると滑らかになりますが、PC負荷が増えます。 必要サンプル数は下記のPrint Informationを参考にしてください。                 |
| **Preview Limit**     | プレビュー時の描画精度を変更し、編集を軽くできます。0で無効。                     |
| **Extrapolation**     |  0フレームより前の動きを仮想的に補間します。<br>・None：補間しない<br>・Linear：1次補間<br>・Quadratic：2次補間          |
| **Resize**            | ブラーが見切れないように、描画範囲を自動で拡大します。                                |
| **Geo Cache**         | スクリプトによる座標データを保存するか指定します。<br>・None：保存しない<br>・Full：全フレーム保存<br>・Minimal：必要最低限のみ保存                     |
| **Cache Control**     | ジオメトリデータを削除するタイミングを指定します。<br>・Off：削除しない<br>・Auto：オブジェクトの最終フレームで削除<br>・All：次スクリプトが読み込まれた時すべて削除<br>・Current：次スクリプトが読み込まれた時現在オブジェクトのみ削除 |
| **Mix**               | 元の画像を重ねて残像を作ります。値を上げると残像感が強くなります。                                |
| **Print Information** | ログ(表示→ログを表示)に処理情報（必要サンプル数など）を表示します。                               |
