# 角丸補正_Rh

ぼかしを用いてオブジェクトのフチをなめらかにし、角を「丸く見える形」に整えるスクリプト

## 使い方
オブジェクトの追加ボタン → アニメーションの効果 → 角丸補正_Rh

## 項目一覧

| 項目名                | 説明                                                                  |
| :----------------- | :------------------------------------------------------------------ |
| 丸め半径（radius）       | 角を丸める強さ。値が大きいほど角が丸くなる。                                 |
| 太さ補正（fatness）      | 丸め後の「太り/痩せ」を調整。<br>・＋：太め<br>・－：細め                    |
| アンチエイリアス（softness） | フチの滑らかさ。上げるほどギザギザが減る（上げすぎるとぼやけやすい）。                            |
| mode               | 透明度（アルファ）の扱い方。<br>・標準：通常のアルファで処理<br>・メタ：アルファを変換して処理 |
| 元画像表示（sharp）       | 色を元画像寄りにするか。<br>・オン：色は元画像を優先しつつ丸める<br>・オフ：ぼかし結果の色も使う       |
