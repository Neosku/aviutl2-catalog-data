# LineGlitch_Rh

帯状に画面をランダムにずらし、色ずれを加えることで、信号が乱れたような**グリッチ**っぽい表現を作るスクリプト

## 使い方
オブジェクトの追加ボタン（フィルター効果を追加） → アニメーションの効果 → LineGlitch_Rh


## 項目

| 項目名              | 説明                                                                                                  |
| :--------------- | :-------------------------------------------------------------------------------------------------- |
| ズレ強度  | ラインがずれる量の大きさ。                                                                           |
| カラーシフト   | 色ずれの強さ。大きいほど色のフチがずれる。                                                           |
| 色ずれ種類  | 色ずれの組み合わせ（1〜3）。<br>・**1**：RとBが逆方向にずれる（Gは基準）<br>・**2**：GとBが逆方向にずれる（Rは基準）<br>・**3**：RとGが逆方向にずれる（Bは基準） |
| 確率  | ずらすラインの割合（0〜100%）                                                 |
| 基準        | ずれの“寄り”を調整（-100〜100）。<br>・マイナス：左（または上）に寄りやすい<br>・プラス：右（または下）に寄りやすい<br>※「縦方向」をONにすると上下方向の寄りになる       |
| 幅          | ずれを適用する帯の太さ（ピクセル単位の目安）。<br>・小さい：細かいラインが多い<br>・大きい：太い帯でずれる                                           |
| ノイズ変化     | グリッチのパターンを毎フレーム変化させる                                               |
| 縦方向    | ずれる向きを縦方向に切り替える                                              |
| 領域拡張     | ずれで端が欠けないように、必要なぶんだけ表示領域を広げる|
| 端部透過 | ずれで画面外を参照した部分の扱い。<br>・ON：端を透明にする<br>・OFF：端を黒で埋める                                           |
