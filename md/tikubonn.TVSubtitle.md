# テレビ字幕スクリプト

テキストオブジェクトに適用することで、
テレビ字幕のように、テキスト裏に背景を追加するスクリプトです。
余白サイズ・背景色・合成モードの変更など、様々な機能が用意されています。

> [!NOTE]
> 本スクリプトはテキストオブジェクト向けに設計されているだけなので、
> 図形・画像オブジェクトなどでも利用することができます。

## 提供されるアニメーション効果

### テレビ字幕@テレビ字幕

テレビ字幕のようにテキスト裏に背景を描画します。

#### パラメータ

- **余白X, 余白Y**
  描画されるテキスト背景の追加の余白幅を設定できます。

- **閾値**
  テキスト領域の検出時のアルファ値に対する感度を設定できます。
  本パラメータは 0 ~ 255 の範囲の値を設定でき、
  値が低ければ低いほど、より小さな不透明度に対して反応し、
  値が大きければ大きいほど、より大きな不透明度にしか反応しないようになります。

  > [!NOTE]
  > 通常用途では弄る必要はありません。

- **接着間隔**
  テキスト領域の検出時に隣り合う領域を統合する間隔を設定できます。
  例えばアルファベットの i, j などは上部の点が本体から離れているため、
  未設定のままだと点と本体が別のテキスト領域として検出されてしまいます。
  本パラメータを用いることでそれらを1つの領域として統合することができます。

- **透明度**
  描画背景の透明度を表す 0 ~ 100 までの整数です。
  0 ならば不透過 100 ならば透明になります。

- **合成モード**
  描画背景の合成モードを設定できます。
  AviUtl で利用できる大体の合成モードに対応しています。

- **背景色**
  描画背景の色です。
