# 揃えスクリプト

オブジェクトの揃え位置を指定するスクリプトです。
本機能を利用することで、左上揃えのテキストオブジェクトを、画面中央にレイアウトするといったことができるようになります。

## 提供されるアニメーション効果

本スクリプトは揃え位置の指定に 0 ~ 8 までの数値を使用します。
それらの数値と対応する実際の位置関係は下記の表のようになっております。

```txt
0 -- 1 -- 2
|    |    |
3 -- 4 -- 5
|    |    |
6 -- 7 -- 8
```

### 揃え@揃えv2

基本的な揃え機能を提供します。
    
#### パラメータ

- **位置**
  オブジェクトが配置される位置を表番号から指定します。
  例えばオブジェクトを中央配置するならば 4 を指定します。

- **拡大率**
  揃えるオブジェクトの拡大率を変更します。
  字幕のテキストなど、複数一括で設定するならば、グループ制御の拡大率の使用を推奨します。
  本パラメータが追加された経緯については FAQ に記載された "標準描画の拡大率を設定すると上手く表示されない…" をご参照ください。

- **X拡大率**

  字幕のテキストなど、複数一括で設定するならば、グループ制御の拡大率の使用を推奨します。
  本パラメータが追加された経緯については FAQ に記載された "標準描画の拡大率を設定すると上手く表示されない…" をご参照ください。

- **Y拡大率**
  揃えるオブジェクトのY軸の拡大率を変更します。
  字幕のテキストなど、複数一括で設定するならば、グループ制御の拡大率の使用を推奨します。

### 詳細揃え@揃えv2

X,Y軸の揃え位置を -1.0 ~ 1.0 の範囲で指定する機能を提供します。

> [!Note]
> こちらは試験的に実装された機能であり、
> 単にオブジェクトを揃えたいだけならば\[揃え@揃えv2\]の使用を推奨します。

#### パラメータ

- **X**
  X軸の揃え位置を指定します。
  このパラメータは -1.0 ~ 1.0 までの数値を指定します。
  値が -1.0 に近ければ近いほど、オブジェクトは右揃えになり、
  逆に値が 1.0 に近ければ近いほど、オブジェクトは左揃えに近付いていきます。

- **Y**
  Y軸の揃え位置を指定します。
  このパラメータは -1.0 ~ 1.0 までの数値を指定します。
  値が -1.0 に近ければ近いほど、オブジェクトは下揃えになり、
  逆に値が 1.0 に近ければ近いほど、オブジェクトは上揃えに近付いていきます。

- **拡大率**
  揃えるオブジェクトの拡大率を変更します。
  字幕のテキストなど、複数一括で設定するならば、グループ制御の拡大率の使用を推奨します。
  本パラメータが追加された経緯については FAQ に記載された "標準描画の拡大率を設定すると上手く表示されない…" をご参照ください。

- **X拡大率**
  揃えるオブジェクトのX軸の拡大率を変更します。
  字幕のテキストなど、複数一括で設定するならば、グループ制御の拡大率の使用を推奨します。
  本パラメータが追加された経緯については FAQ に記載された "標準描画の拡大率を設定すると上手く表示されない…" をご参照ください。

- **Y拡大率**
  揃えるオブジェクトのY軸の拡大率を変更します。
  字幕のテキストなど、複数一括で設定するならば、グループ制御の拡大率の使用を推奨します。

## FAQ

- **標準描画の拡大率を設定すると上手く表示されない…**
  残念ながら仕様です。
  私の環境で色々試してみましたが、オブジェクトの拡大率、または適用後のサイズを取得することができませんでした。
  そのため、どうしても拡大率を変更したい場合には、揃え・詳細揃えに内蔵された拡大率パラメータを利用するか、グループ制御による操作が必要になります。

- **バグ報告したいんだけど**
  親しい間柄でない限りバグ報告は受け付けておりません。
  それでも報告したい方はこれらの条件を読んだうえで報告してください。

  受け付けるバグの条件は次のとおりです。
  これらの条件を満たさないバグ報告は無視いたします。

  - 本スクリプトの最新版で発生したバグであること
  - 本スクリプトのみを導入した環境で再現できるバグであること
  - 本スクリプトの対応環境で発生したバグであること

  報告をする際には必ずこれらの情報を提供してください。
  
  - 具体的な症状
  - 発生する条件
  - 再現可能なプロジェクトファイル(外部ファイルを参照せずに、図形オブジェクトを使用したもの)
  - 使用している AviUtl, 拡張編集 のバージョン
