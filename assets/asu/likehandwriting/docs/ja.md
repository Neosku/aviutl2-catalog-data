## 概要
手書き風にイラストやテキストをカクカク動かすことのできるスクリプトです。
詳しくは[AVILITY](https://avilityyy.blogspot.com/2026/04/likeHandWriting.html)をご覧ください。

## 使い方
手書き風に動かしたい画像に「手書き風カクカク振動」をつけます。デフォルトでは「装飾」に分類されます。

## パラメーター一覧
| グループ | パラメーター名 | 説明 |
| :---: | :--- | :--- |
| 共通設定 | 間隔 [f] | 変形が切り替わる間隔(フレーム単位) |
| 〃 | シード | 乱数のパターンを決定します。<br>負の値だとオブジェクトに依存せず同一のシードであれば同一のアニメーションになります。 |
| ポリゴン変形 | 方式 (ポリゴン) | メッシュ歪みの種類を選択。<br>「端も変形」は全体が揺れ、「端は変形しない」は輪郭線を維持して中身が歪みます。 |
| 〃 | 横/縦 分割数 | 画像を分割する細かさ。 |
| 〃 | 変化度合 | ポリゴン頂点の移動距離。<br>値が大きいほどクシャッとした見た目になります。 |
| なめらかな変形 | 方式 (なめらか) | エフェクトによる歪み。「ラスター変形」とノイズに基づく「ディスプレイスメントマップ変形」から選択。<br>ノイズの方が不規則な見た目の変化になります。 |
| 〃 | 程度 | 値が大きいほど歪みが強くなります。 |
| 〃 | 横/縦 粗さ | 変形の周期（波の大きさ）。値を大きくすると、よりゆったりとした大きな歪みになります。 |
| 座標・拡大変形 | ☑ 座標・拡大変形 | チェックを入れると、オブジェクトの位置とサイズが「間隔 [f]」に合わせてランダムに振動します。 |
| 〃 | X/Y 座標 | 位置移動の最大距離。座標自体が震えるような「手ブレ」効果を与えます。 |
| 〃 | 拡大率 X/Y | サイズの変動幅。拡大率を抑制にチェックを入れると、元のサイズを超えない範囲で小さく震えます。 |
