アニメーションのmp4やgifにおいて、特定のフレームをループさせることができるスクリプトです。
ループさせても目立たない箇所をループさせることでアニメーションの長さを調整しやすくします。
音声には非対応です。
詳細はコモンズIDの動画(sm45249838)を参照してください。
設定値に関する説明は、動画説明欄の配布URLからダウンロードしたzipファイル内のpdfにあります。

[親作品登録について]
使用した場合の親作品登録については任意です。
親作品登録していただけると利用状況が分かりありがたいです。

[注意]
入力プラグインにおいて、gifは標準のWIC image file readerではなくL-SMASH Worksを使用してください。
標準のWIC image file readerではフレーム数の情報が取得できず正しく動作しません。
